Artifact Overview

The artifact selected for this enhancement is my CS 360 Weight Tracking Android Application, which was originally created during my mobile architecture and programming course. The application allows users to create an account, record weight entries, set a target weight, and track their progress over time.

I selected this artifact because it demonstrates my ability to develop a functional mobile application that combines user input, multiple screens, database storage, validation, and progress tracking. It also provided several opportunities to improve the design, usability, and reliability of an existing software system.

Software Design Enhancement

For the software design and engineering enhancement, I improved the application from a basic weight-entry tool into a more organized and user-friendly application. I redesigned the home screen so the dashboard clearly displays the user's current weight, target weight, most recent entry, and progress toward the selected goal.

I also improved the navigation between the login, home, and goal-setting screens while making the buttons and actions more consistent. Stronger input validation was added to prevent users from saving empty, negative, or unrealistic weight values. Record-editing and deletion features were also improved so users can manage existing information without creating inaccurate or duplicate entries.

Skills Demonstrated

This enhancement demonstrates my skills in Android application development, interface design, input validation, debugging, software organization, and user-focused development. It also demonstrates my ability to evaluate an existing application, identify weaknesses in its structure and workflow, and implement changes that make the software more reliable and useful.

Course Outcome

This enhancement supports the course outcome of using well-founded techniques, skills, and computing tools to implement solutions that deliver value and accomplish practical goals. The redesigned dashboard, improved navigation, and stronger validation provide clearer feedback and make the application easier for users to understand and operate.

Reflection

While completing this enhancement, I learned that software design involves more than changing the appearance of an application because the interface, navigation, stored data, and user feedback must work together consistently. One challenge was ensuring that the dashboard refreshed correctly after users added, edited, deleted, or updated information, which required changes to both the interface logic and database interactions.
