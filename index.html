Professional Self-Assessment

Completing the Computer Science program and developing this ePortfolio have helped me recognize my strengths, clarify my professional goals, and prepare for a career in information technology or software development. Throughout the program, I gained experience in programming, mobile application development, databases, cybersecurity, full-stack development, testing, and problem-solving, which helped me become more confident in my ability to design, improve, and explain technical solutions. My coursework also strengthened my ability to collaborate, communicate with different audiences, and consider the needs of users. Through project reports, diagrams, code reviews, and technical explanations, I learned how to present complex information clearly while also making practical decisions about software design. I developed experience with data structures and algorithms by organizing, sorting, and processing information efficiently, while my software engineering and database coursework taught me how to build reliable applications, manage stored data, and improve existing systems. Security became an important part of my approach to software development because I learned that user data and system weaknesses must be considered throughout the design process. In addition to the artifact in this portfolio, I completed projects involving full-stack development, embedded systems, secure coding, input validation, SQL injection prevention, encryption, static analysis, and unit testing, which helped me understand how different areas of computer science work together. The main artifact in this portfolio is my CS 360 Weight Tracking Android Application, which allows users to create an account, record their weight, set a goal, and track their progress. I improved the application through three connected enhancements involving software design and engineering, algorithms and data structures, and databases, which together demonstrate my ability to improve usability, organize data efficiently, protect user information, and create a more reliable application. These enhancements, along with the code review and written narratives, show my growth throughout the Computer Science program and introduce the technical work presented in the rest of my ePortfolio.

Featured Artifact

The featured artifact is a Weight Tracking Android Application that allows users to create an account, record their weight, set a goal, and review their progress.

Enhancement Categories

Software Design and Engineering

Improved the dashboard, navigation, validation, goal-setting, and record-editing features.

Algorithms and Data Structures

Organized weight records as structured objects and sorted them from newest to oldest.

Databases

Separated user, weight, and goal information into organized tables and replaced plain-text password storage with salted password hashing.
